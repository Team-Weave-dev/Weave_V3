## 참고사항

### 인터뷰 진행 팁

1. **중립적 태도 유지**
   - "좋은데요?"와 같은 유도 질문 지양
   - "어떠셨어요?"와 같은 열린 질문 사용

2. **Think Aloud 유도**
   - "지금 무엇을 보고 계신가요?"
   - "무엇을 하려고 하시나요?"
   - "왜 그렇게 생각하셨어요?"

3. **침묵 활용**
   - 참여자가 생각할 시간 충분히 제공
   - 바로 다음 질문으로 넘어가지 않기

4. **비언어적 신호 관찰**
   - 표정, 제스처, 한숨, 웃음 등 기록
   - 말과 행동의 불일치 체크

5. **깊이 있는 질문**
   - "왜 그렇게 생각하셨어요?"
   - "조금 더 자세히 말씀해 주시겠어요?"
   - "예를 들면 어떤 상황인가요?"

### 문제 상황 대처

**참여자가 막혔을 때**:
- 즉시 도와주지 말고 관찰 (30초-1분)
- "어떤 부분이 어려우신가요?" 질문
- 힌트 제공 순서: 최소 → 중간 → 직접 시연

**기술적 문제 발생 시**:
- 타임스탬프 기록
- 문제 재현 시도
- 대안 제시 (백업 환경)

**시간 초과 시**:
- 중요도 낮은 질문 생략
- 핵심 질문 우선 진행
- 참여자 동의 하에 연장
