## 디렉토리 목적
shadcn/ui 기반 공통 컴포넌트와 확장 위젯을 제공합니다.
버튼, 입력, 테이블 등 기본 요소를 프로젝트 스타일에 맞게 커스터마이즈합니다.

## 핵심 책임
- 기본 UI 컴포넌트 스타일 확장
- 접근성과 반응형 규칙 유지
- 고급 위젯의 기반 구성 제공
